GPU Guide

Everything you need to know about graphics cards before you buy.

What is a GPU?

A GPU (Graphics Processing Unit) is a chip dedicated to rendering images, video, and animations on your screen. Unlike a CPU, which handles a few complex tasks at once, a GPU excels at running thousands of smaller tasks in parallel — exactly what rendering 3D graphics, ray-traced lighting, and high-resolution video requires. For gaming, the GPU is almost always the single biggest factor in how many frames per second you get. For creative work like 3D rendering or video editing, a faster GPU dramatically cuts down processing time.

Key Specs to Look For

VRAM

Video RAM is the GPU's dedicated memory for textures, frame buffers, and assets. Running out of VRAM causes stutters and crashes.

  • 8 GB — minimum for 1080p/1440p gaming today
  • 12 GB — comfortable for 1440p, modded games, some AI
  • 16 GB+ — 4K gaming, video editing, ML workloads

Clock Speed

Measured in MHz. Base clock is the minimum sustained speed; boost clock is the peak the GPU hits under load. A higher boost clock generally means better performance, but architecture matters more than raw clock speed.

TDP / Power Draw

Thermal Design Power is how many watts the GPU consumes at peak. This directly determines what PSU you need. High-end GPUs (RTX 4090: 450W) require 850W+ PSUs with multiple PCIe power connectors. Always add 100–150W of headroom above your total system power draw.

Memory Bandwidth

How fast data moves between the GPU and its VRAM, measured in GB/s. Higher bandwidth reduces bottlenecks when textures and frame data need to be fetched quickly. This is especially important at 4K where more data is processed per frame.

Architecture

The GPU's underlying design generation. Newer architectures bring better performance-per-watt, improved ray tracing, and new features like upscaling. Current generations:

  • NVIDIA — Ada Lovelace (RTX 40 series), Blackwell (RTX 50 series)
  • AMD — RDNA 3 (RX 7000 series), RDNA 4 (RX 9000 series)
  • Intel — Xe-HPG / Arc Battlemage (Arc B-series)

Common Mistakes When Buying a GPU

  • CPU bottlenecking — Pairing a high-end GPU with a very old or weak CPU means the CPU can't feed the GPU fast enough, wasting GPU performance. Match your CPU tier to your GPU tier.
  • Not checking PSU wattage — A high-end GPU on an underpowered PSU can cause crashes, instability, or even hardware damage. Always check GPU TDP and leave 150W+ headroom in your PSU rating.
  • Ignoring physical size — Flagship GPUs can be 330–360mm long and 3+ slots thick. Confirm your case supports the length and that nothing blocks the PCIe slot area before buying.
  • Overpaying for VRAM you won't use — VRAM matters, but 16 GB is overkill for most 1440p gaming builds today. Don't pay a large premium for VRAM headroom you won't realistically need for 2–3 years.

CPU Guide

Everything you need to know about processors before you buy.

What is a CPU?

A CPU (Central Processing Unit) is the brain of your PC — it handles every instruction your software sends, from running the operating system to executing game logic and processing physics. Unlike a GPU which runs thousands of tiny parallel tasks, a CPU excels at running a smaller number of complex, sequential tasks very fast. For gaming, a strong single-core speed matters most; for creative work like video editing or 3D rendering, having more cores significantly reduces processing time.

Key Specs to Look For

Core Count

The number of physical processing units. More cores help with multitasking and parallel workloads, but most games still rely more on single-core speed.

  • 6 cores — fine for gaming, light productivity
  • 8–12 cores — gaming + streaming, video editing
  • 16+ cores — heavy rendering, compiling, professional workloads

Clock Speed

Measured in GHz. Base clock is the default operating speed; boost clock is the maximum speed reached under single-core load. A higher boost clock generally means snappier single-threaded performance, which matters most for gaming.

TDP / Power Draw

How many watts the CPU consumes at peak load. Higher-end CPUs (125W+) generate more heat and require a capable cooler — stock coolers are often insufficient. TDP also affects your total system power budget when sizing a PSU.

Cache (L3)

On-chip memory that stores frequently used data for ultra-fast access. A larger L3 cache reduces latency for gaming workloads. AMD's 3D V-Cache technology dramatically increases cache size and measurably improves gaming frame rates.

Socket & Platform

The CPU socket must match your motherboard. Sockets are not interchangeable between brands or generations.

  • AMD AM5 — Ryzen 7000/9000 series, DDR5
  • Intel LGA1700 — 12th–14th Gen, DDR4 or DDR5
  • Intel LGA1851 — Core Ultra 200 series, DDR5

Memory Support

CPUs specify which DDR generation they support (DDR4 or DDR5) and the maximum officially supported speed. Running RAM at its rated speed (via XMP/Expo) can noticeably improve performance, especially on AMD Ryzen platforms.

Common Mistakes When Buying a CPU

  • Chasing core count for gaming — More cores don't automatically mean more FPS. Most games are still limited by single-core speed. A 6-core CPU with a high boost clock will outperform a 16-core CPU with a slow one in most titles.
  • Wrong socket for your motherboard — Intel and AMD use completely different sockets, and even within a brand, generations don't always share sockets. Always confirm CPU–motherboard socket compatibility before buying.
  • Skipping XMP/Expo — By default, most motherboards run RAM at 2133–4800 MHz regardless of what speed it's rated for. Enable XMP (Intel) or Expo (AMD) in the BIOS to unlock your RAM's full rated speed, which can meaningfully improve performance.
  • Undercooling a high-TDP chip — CPUs rated at 125W+ will throttle and underperform on a cheap or stock cooler. Match your cooler's TDP rating to your CPU, and for overclocked builds, go even higher.

Motherboard Guide

Everything you need to know about motherboards before you buy.

What is a Motherboard?

A motherboard is the main circuit board that connects every component in your PC — the CPU, RAM, GPU, storage, and power supply all plug into or communicate through it. It determines which CPUs and RAM are compatible with your build, how many storage drives you can connect, and what expansion options you have. Choosing the right motherboard means matching it to your CPU socket, your use case, and your budget — a more expensive board doesn't always mean better performance, but it does unlock features like overclocking, more USB ports, and better power delivery.

Key Specs to Look For

CPU Socket

The physical slot the CPU locks into. Socket must exactly match your CPU — there is no cross-compatibility.

  • AM4 — AMD Ryzen 1000–5000 series
  • AM5 — AMD Ryzen 7000/9000 series
  • LGA1700 — Intel 12th–14th Gen
  • LGA1851 — Intel Core Ultra 200 series

Chipset

Controls which features are available: overclocking support, number of PCIe lanes, USB ports, and M.2 slots. Higher chipsets cost more but unlock more capability.

  • AMD: A620 → B650 → X670/X870 (ascending capability)
  • Intel: H610 → B760 → Z790 (Z boards allow CPU OC)

Form Factor

The physical size of the board — determines what cases it fits in and how many slots it has.

  • ATX — Full-size, most slots, fits mid/full-tower cases
  • Micro ATX — Smaller, fewer slots, fits most cases
  • Mini ITX — Compact, limited slots, small form factor cases only

RAM Slots & Support

Most ATX boards have 4 DIMM slots; Micro ATX usually 4; Mini ITX typically 2. Check the maximum supported RAM capacity and speeds. Always verify your RAM kit is on the motherboard's QVL (qualified vendor list) for guaranteed compatibility.

M.2 & SATA Slots

M.2 slots are for fast NVMe SSDs. More slots = more high-speed storage. SATA ports connect older SSDs and HDDs. Budget boards may have 1–2 M.2 slots; high-end boards can have 4+. Check whether slots support PCIe Gen 4 or Gen 5 for maximum SSD speed.

VRM & Power Delivery

The Voltage Regulator Module (VRM) controls power to the CPU. A stronger VRM is essential for overclocking or running high-TDP CPUs. Budget boards with weak VRMs can throttle or overheat under sustained load with powerful processors.

Intel LGA1700 / LGA1851
  • Wide range of chipset and price options
  • DDR4 support on LGA1700 (lower platform cost)
  • B760 boards offer great value for non-OC builds
  • Z790 unlocks full CPU overclocking
  • Broad availability from many manufacturers
LGA1700 platform aging out; LGA1851 is DDR5-only and newer/pricier
Best for: budget builds (B760 + DDR4), established platform reliability
AMD AM5 (B650 / X670 / X870)
  • AM5 platform intended to last multiple CPU generations
  • PCIe 5.0 on X670/X870 for next-gen SSDs and GPUs
  • B650 boards hit a solid mid-range price point
  • Expo support for easy DDR5 overclocking
  • Strong feature set even on mid-tier chipsets
DDR5-only increases upfront platform cost; no DDR4 option on AM5

Common Mistakes When Buying a Motherboard

  • Wrong socket for your CPU — This is the most common and most expensive mistake. An AMD Ryzen 7000 CPU needs an AM5 board; it will not fit an AM4 board. Always double-check socket compatibility before purchasing.
  • Overlooking VRM quality for powerful CPUs — A cheap B-series board with a weak VRM paired with a 125W+ CPU will throttle under sustained load. If you're running a high-end CPU, spend appropriately on the board's power delivery.
  • Not checking M.2 slot sharing — Some motherboards disable SATA ports or share PCIe bandwidth when certain M.2 slots are occupied. Read the manual's "bandwidth sharing" section before planning your storage layout.
  • Buying a Z-series board without overclocking — Z790 and X670 boards are great, but if you don't plan to overclock, a B760 or B650 board at half the price delivers nearly identical real-world performance.

RAM Guide

Everything you need to know about system memory before you buy.

What is RAM?

RAM (Random Access Memory) is your PC's short-term working memory. When you open a game, browser, or application, it loads data from your storage drive into RAM so the CPU can access it instantly — storage is far too slow for real-time use. More RAM means more things can run at once without slowdown; faster RAM means data gets to the CPU more quickly. Unlike storage, RAM is volatile — everything in it is cleared when the PC powers off.

Key Specs to Look For

Capacity

How much data RAM can hold at once. More capacity lets you run more programs simultaneously without the OS swapping to slower storage.

  • 8 GB — bare minimum, tight for modern gaming
  • 16 GB — recommended minimum for gaming today
  • 32 GB — gaming + streaming, creative work
  • 64 GB+ — video editing, 3D rendering, VMs

Speed (MHz)

How many data transfers per second the RAM can perform. Faster RAM reduces latency between CPU and memory. AMD Ryzen is especially sensitive to RAM speed — running at the rated speed (via Expo) can noticeably improve gaming FPS.

  • DDR4: sweet spot at 3200–3600 MHz
  • DDR5: sweet spot at 6000–6400 MHz

DDR Generation

DDR4 and DDR5 are not interchangeable — the physical slots are different. Your CPU and motherboard dictate which generation you use. DDR5 is faster and the current standard for new platforms, but DDR4 is cheaper and still very capable.

Timings (Latency)

Expressed as numbers like CL16 or CL30. Lower CL numbers mean faster response. Timings and speed work together — a DDR5-6000 CL30 kit can outperform a DDR5-6400 CL36 kit. Don't focus on speed alone.

Dual Channel

Using two matched sticks instead of one activates dual-channel mode, effectively doubling the memory bandwidth available to the CPU. This is one of the easiest free performance gains — always use RAM in pairs (e.g., 2×16 GB instead of 1×32 GB).

XMP / Expo

RAM ships running at a conservative default speed (often 2133–4800 MHz) regardless of its rated speed. XMP (Intel) and Expo (AMD) are BIOS profiles that unlock the rated speed with one toggle. Always enable this after building — it's safe and often free performance.

DDR4 vs DDR5

DDR4 Previous Generation
  • Significantly cheaper per GB
  • Lower latency at equivalent speeds
  • Mature, widely compatible ecosystem
  • Still very capable for gaming and productivity
  • Available on Intel LGA1700 platforms

DDR5 Current Generation
  • Higher peak bandwidth and speeds
  • Required for AM5 (AMD) and LGA1851 (Intel)
  • Better performance scaling at high core counts
  • On-die ECC for improved reliability
  • Future-proof for new platforms
Higher price per GB; higher base latency than DDR4

Common Mistakes When Buying RAM

  • Not enabling XMP/Expo — This is the most common mistake. RAM defaults to a slow speed out of the box. If you don't enable XMP or Expo in the BIOS, your expensive 6000 MHz kit runs at 4800 MHz. One toggle in BIOS fixes it.
  • Buying a single stick instead of a kit — A single 32 GB stick runs in single-channel mode, cutting your memory bandwidth roughly in half versus two 16 GB sticks. Always buy matched kits and install them in the correct paired slots (usually A2/B2).
  • Mixing RAM kits — Adding a second kit from a different brand, speed, or batch can cause instability or force both kits to run at the slower speed. Buy all your RAM at once from the same kit if possible.
  • Overkill capacity for gaming — 64 GB of RAM gives no gaming performance benefit over 32 GB. Spend the difference on a better GPU or CPU where it actually shows up in frame rates.

PSU Guide

Everything you need to know about power supplies before you buy.

What is a PSU?

A Power Supply Unit (PSU) converts AC power from your wall outlet into the stable DC voltages your PC components need to run. Every component in your build — CPU, GPU, storage, fans — draws power through the PSU. Choosing the right one means matching wattage to your system's demands, picking a reliable unit with a good efficiency rating, and leaving enough headroom so the PSU isn't running at its limit. A low-quality or underpowered PSU is one of the most common causes of system crashes, instability, and in extreme cases, component damage.

Key Specs to Look For

Wattage

The total continuous power the PSU can deliver. Add up your CPU TDP + GPU TDP + ~100W for the rest of the system, then add 20–30% headroom.

  • 550–650W — budget/mid-range builds (RTX 4060 tier)
  • 750–850W — mid-high builds (RTX 4070–4080 tier)
  • 1000W+ — flagship GPUs (RTX 4090 / RX 7900 XTX)

80 Plus Efficiency Rating

Measures how efficiently the PSU converts AC to DC. A higher rating wastes less power as heat, runs cooler, and costs less to operate. The rating tiers from lowest to highest:

  • 80+ White — 80% efficiency, avoid if possible
  • 80+ Bronze — acceptable for budget builds
  • 80+ Gold — recommended sweet spot
  • 80+ Platinum / Titanium — premium efficiency

Modularity

Determines which cables are permanently attached to the PSU.

  • Non-modular — all cables permanently attached; cheap but messy
  • Semi-modular — essential cables fixed, rest detachable; good balance
  • Fully modular — all cables detachable; best cable management, higher cost

Voltage Rails

Modern PSUs deliver power primarily on a single +12V rail. The 12V rail powers the CPU and GPU — the components with the highest power demand. Check that the PSU's 12V amperage rating matches the combined draw of your CPU and GPU, not just total wattage.

Protection Features

Quality PSUs include hardware protections that cut power before damage occurs:

  • OVP — Over Voltage Protection
  • OCP — Over Current Protection
  • OTP — Over Temperature Protection
  • SCP — Short Circuit Protection

Form Factor

Most desktop builds use ATX PSUs. Small form factor builds may require SFX or SFX-L units, which are physically smaller but often cost more per watt. Always check your case's PSU size spec before buying.

Common Mistakes When Buying a PSU

  • Buying a no-name or unrated PSU to save money — A cheap PSU is the one place you should never cut corners. A failing PSU can send voltage spikes through your system and kill your CPU, GPU, or motherboard. Stick to established brands with at least an 80+ Bronze rating.
  • Underestimating wattage requirements — Running a PSU at 90–100% of its rated capacity stresses it, shortens its lifespan, and causes voltage instability. Always leave at least 20% headroom. For an RTX 4090 build, a 850W PSU is not enough — go 1000W or higher.
  • Ignoring the 12VHPWR connector for new GPUs — RTX 40/50 series and RX 7900 cards may use the 16-pin 12VHPWR connector. Ensure your PSU either includes this cable natively or that the adapter is rated for your GPU's power draw. Using a poorly seated adapter has caused GPU fires.
  • Buying more wattage than needed — A 1600W PSU on a 300W system wastes money and runs less efficiently at low load. Right-size your PSU — more headroom is good, but excessive headroom just costs more without benefit.

CPU Cooler Guide

Everything you need to know about CPU cooling before you buy.

What is a CPU Cooler?

A CPU cooler draws heat away from the processor and dissipates it into the surrounding air. CPUs generate significant heat under load — without a cooler, a modern CPU would overheat and shut down within seconds. The cooler sits directly on top of the CPU, connected via a thin layer of thermal paste that fills microscopic gaps and improves heat transfer. A better cooler keeps temperatures lower, which allows the CPU to sustain higher boost clocks for longer, reduces fan noise, and extends the component's lifespan.

Key Specs to Look For

TDP Rating

Thermal Design Power — the maximum heat load the cooler is rated to handle, in watts. Match or exceed your CPU's TDP rating. For overclocked or high-end CPUs, go higher than the rated TDP to give yourself thermal headroom.

  • 65W CPU — budget air cooler is sufficient
  • 105–125W CPU — mid-range tower or 240mm AIO
  • 170W+ CPU — high-end tower or 280/360mm AIO

Noise Level (dBA)

Measured in decibels. Lower is quieter. Most coolers list noise at maximum fan speed — real-world use is often much quieter since fans rarely hit 100% under normal gaming loads. Anything under 35 dBA is considered quiet; under 25 dBA is near-silent.

Cooler Height

Air tower coolers can be 150–170mm tall. Check your case's maximum CPU cooler clearance before buying — a cooler that's too tall simply won't fit with the side panel on. Low-profile coolers (under 60mm) are available for compact builds.

Socket Compatibility

Coolers mount via a bracket that must match your CPU socket. Most modern coolers support both Intel (LGA1700, LGA1851) and AMD (AM4, AM5), but always verify before purchasing. AM5 in particular has seen some cooler compatibility issues with older mounting hardware.

Fan Speed (RPM)

Higher RPM moves more air but creates more noise. Good coolers use PWM (Pulse Width Modulation) fans that automatically adjust speed based on CPU temperature — fast when needed, near-silent at idle. Look for PWM support when comparing fans.

Radiator Size (AIO)

For All-in-One liquid coolers, the radiator size determines cooling capacity. Larger radiators have more surface area to dissipate heat.

  • 120/240mm — mid-range CPUs, compact cases
  • 280mm — strong cooling, slightly smaller than 360
  • 360mm — high-end CPUs, maximum thermal headroom

Common Mistakes When Buying a CPU Cooler

  • Using the stock cooler on a high-TDP CPU — Stock coolers are designed to barely meet thermal limits, not exceed them. On a 125W+ CPU under sustained load, a stock cooler will throttle the CPU to protect it, reducing performance. An aftermarket cooler is always worth it.
  • Not checking case clearance — A 165mm air tower in a case with 155mm of clearance means the side panel won't close. Always check your case's maximum CPU cooler height spec before buying a large tower cooler.
  • Skipping thermal paste application — Most coolers come with pre-applied or included thermal paste. If you're reusing a cooler or replacing it, clean off old paste completely and apply a fresh pea-sized amount. Dried or improperly applied paste significantly raises temperatures.
  • Assuming bigger AIO always beats air — A high-end dual-tower air cooler like the Noctua NH-D15 consistently matches or beats 240mm AIOs in temperature. A 240mm AIO is not a guaranteed upgrade over a good air cooler — often it's a lateral move that costs more.

Builder Tips

Always use XMP/Expo

RAM runs at low default speeds unless you enable XMP (Intel) or Expo (AMD) in BIOS.

Don't skip thermal paste

A pea-sized dot on the CPU center is all you need. Too much is worse than too little.

Monitor goes in the GPU

First-time builders often plug into the motherboard port — always plug into the GPU's output.

Overspec your PSU by 20%

If your build needs 400W, get a 550W unit. Headroom = efficiency and longevity.

Install M.2 before the motherboard

It's nearly impossible to reach M.2 slots after the board is mounted in the case.

Dual channel matters

Two 8GB sticks outperform one 16GB stick. Always buy RAM in matched pairs.

NVMe > SATA SSD > HDD

For your OS drive, get at least a SATA SSD. NVMe is only meaningfully faster for large file transfers.

Don't build on carpet

Static discharge can kill components. Build on a hard flat surface and ground yourself regularly.

How to Build a PC

A step-by-step guide for first-time builders.

Before You Start

Gather your parts and prepare your workspace. You'll need a Phillips-head screwdriver, a flat non-carpeted surface (to avoid static), and about 2–3 hours. Anti-static precautions are good practice — touch a grounded metal surface periodically or use an anti-static wrist strap.

Step-by-Step Assembly

  1. Install the CPU — Open the motherboard socket lever, align the CPU's triangle marker with the socket corner, lower it gently, and lock the lever. Never force it — CPUs drop in with zero insertion force.
  2. Install RAM — Match the notch on the stick to the slot. Press firmly until both side clips snap in. If using 2 sticks, use the A2/B2 slots (usually slots 2 and 4) for dual-channel mode.
  3. Install the M.2 SSD — Slide it into the M.2 slot at a shallow angle, push flat, and secure with the small screw. Do this before mounting the motherboard — it's much easier.
  4. Mount the CPU cooler — Apply a pea-sized amount of thermal paste to the center of the CPU if none is pre-applied. Secure the cooler using its mounting bracket. Don't overtighten.
  5. Install the I/O shield — Snap the motherboard's metal I/O shield into the back of the case before mounting the board. It only fits one way.
  6. Mount the motherboard — Install brass standoffs in the case first. Align the motherboard's I/O ports with the shield, then screw it down. Don't cross-thread the screws.
  7. Install the PSU — Slide it into the PSU bay (fan facing down if there's a bottom vent). Secure with 4 screws, then route cables through the case before connecting them.
  8. Install the GPU — Remove the PCIe slot covers from the case. Snap the GPU into the top PCIe x16 slot until it clicks, then secure the bracket screw.
  9. Connect all cables — 24-pin ATX motherboard power, 8-pin CPU power (top of board), PCIe power to the GPU, SATA power for any drives, and front panel connectors (power button, USB, audio headers).
  10. First boot — Connect monitor, keyboard, and power cable. Press the power button. If it posts and reaches BIOS, you're in great shape. Set the boot device to your SSD and install your OS.

Common First-Boot Issues

  • No display — Check that the monitor is plugged into the GPU, not the motherboard. Re-seat the GPU and RAM if no image appears.
  • Won't power on — Verify the front panel power button header is correctly connected. Check the PSU switch is set to ON.
  • Boots but RAM speed is wrong — Enter BIOS and enable XMP (Intel) or Expo (AMD) to run RAM at its rated speed.
  • CPU temps high at idle — The cooler may not be seated properly. Re-apply thermal paste and remount.

Overclocking Basics

How to safely squeeze more performance from your CPU and GPU without spending anything extra.

What is Overclocking?

Overclocking means running a component faster than its factory-rated speed by increasing the clock multiplier or frequency. Modern CPUs and GPUs are usually manufactured with more headroom than advertised — overclocking lets you use it. The trade-off is higher heat and power draw, and in extreme cases, reduced component lifespan if done carelessly.

CPU Overclocking

  • Unlocked CPUs only — AMD Ryzen CPUs and Intel K/KF/KS series are overclockable. A locked Intel Core i5 (no "K") is not. Check your model.
  • Do it in BIOS — Enter BIOS on boot (usually Del or F2). Find "CPU Ratio" or "Core Multiplier." Raise in small steps (e.g., 4.5 GHz → 4.7 GHz), then stress test.
  • Stress test before trusting it — Run Prime95 or Cinebench R23 for 15–30 min. Watch temps with HWiNFO. Keep under 90°C for most CPUs.
  • Voltage (Vcore) — More clock often requires more voltage. Be conservative — going above 1.35V on most CPUs risks long-term degradation. Raise in 0.025V steps only.
  • Don't forget RAM (XMP/Expo) — Enabling XMP (Intel) or Expo (AMD) is technically an overclock and gives free performance. Always enable this first.

GPU Overclocking

GPU overclocking is more approachable than CPU overclocking — the GPU will downclock itself if it gets too hot, so it's harder to brick. Use MSI Afterburner:

  • Raise Power Limit first — Drag the Power Limit slider to +15–20%. This alone can boost boost clocks.
  • Core Clock offset — Add +50–100 MHz to start. Run a benchmark (e.g., Unigine Heaven) and check for artifacts.
  • Memory Clock — VRAM can often take +200–500 MHz. Watch for visual glitches — that's the sign you've gone too far.
  • Fan curve — Edit the fan curve to be more aggressive if temps exceed 85°C under load.

Safety Rules

  • Always stress test after every change — don't assume stable because it booted.
  • Keep a BIOS reset available — know how to clear CMOS (pull battery or short the jumper) in case of a failed overclock.
  • Better cooling = more overclocking headroom. Invest in a better cooler before pushing voltage.
  • Stability at stock is step zero — if your PC crashes at stock speeds, overclocking is not the fix.

Compatibility 101

The essential checklist to make sure every part you buy works with every other part.

CPU ↔ Motherboard

This is the most critical compatibility check. The CPU socket on the motherboard must match the CPU exactly.

  • AMD AM5 — Ryzen 7000/8000/9000 series. Works with B650, X670, X870 chipsets.
  • AMD AM4 — Ryzen 1000–5000 series. Works with B450, X470, B550, X570 chipsets.
  • Intel LGA1700 — 12th, 13th, 14th Gen Core. Works with B660, Z690, Z790 chipsets.
  • Intel LGA1851 — Core Ultra 200 (Arrow Lake). Works with Z890 chipset.
  • Even if the socket matches, check if your board supports that specific CPU generation — a BIOS update may be required for newer CPUs on older boards.

RAM ↔ Motherboard

  • DDR4 vs DDR5 — Motherboards support one or the other, never both. Check your board's spec sheet.
  • Max speed — Your motherboard lists a max supported RAM speed. RAM faster than that will still work but will downclock to the max supported speed.
  • Slots — Most consumer boards have 2 or 4 slots. Check the manual for which slots to use with 2 sticks (usually A2/B2).
  • Capacity — Boards have a max supported RAM amount (e.g., 64GB or 128GB). Check before buying 64GB sticks.

GPU ↔ Case & PSU

  • GPU length — High-end cards can be 340mm+. Check your case's max GPU clearance spec.
  • PSU connectors — Most modern GPUs need 1–3 PCIe power connectors (8-pin or 16-pin 12VHPWR). Verify your PSU has them.
  • Wattage — Check the GPU's TDP plus your CPU's TDP and add 100W of headroom. A 400W total system needs at least a 550W PSU.

Motherboard ↔ Case

  • ATX — Full-size boards (305×244mm). Require a mid-tower or full-tower case.
  • Micro-ATX — Smaller (244×244mm). Fit in most cases, including many compact ones.
  • Mini-ITX — Smallest (170×170mm). Require a specific ITX case and sacrifice expansion slots.
  • Always check the case's supported form factor list — a case built for mATX will not fit a full ATX board.

Storage

  • M.2 NVMe vs SATA — Both use M.2 slots but are not interchangeable on all boards. Check if your M.2 slot supports NVMe, SATA, or both.
  • PCIe generation — PCIe 5.0 drives are backward compatible in PCIe 4.0 slots — they'll just run at PCIe 4.0 speeds. No compatibility issue, just reduced peak speed.

How to Build

Cable Management Tips

Keep your build clean and airflow-efficient with these simple cable management habits.

  • Route cables behind the motherboard tray whenever possible — most modern cases have grommets for this.
  • Use the included velcro straps or zip ties to bundle cables together. Don't overtighten — it can damage cables over time.
  • Connect the 8-pin CPU cable before mounting the motherboard — it's much easier to reach.
  • Measure and tuck excess cable length into unused drive bays or behind the shroud.
  • Group cables by type: power cables together, data cables (SATA) together.
  • Check that no cables are blocking fans or the GPU cooler intake.
Picking Parts

How to Choose a GPU

What to look for when picking a graphics card for gaming, content creation, or AI work.

  • Target resolution first — 1080p: RTX 4060 / RX 7600 tier. 1440p: RTX 4070 / RX 7700 XT tier. 4K: RTX 4080 / RX 7900 XTX tier.
  • VRAM matters — 8 GB is the current minimum for 1440p gaming. 12 GB+ is better for modded games and high-res textures. 16 GB+ for AI or creative workloads.
  • NVIDIA vs AMD — NVIDIA has better ray tracing and DLSS upscaling. AMD cards offer better rasterization performance per dollar and FidelityFX Super Resolution. Both are excellent.
  • Check your PSU wattage — High-end GPUs (RTX 4080/4090) require 850W+ PSUs. Mid-range cards are fine with 650–750W.
  • Measure your case — Flagship GPUs can be 330mm+ long. Confirm your case supports the GPU length.
  • Future-proofing — The next generation is always coming. Buy what fits your budget today; upgrade in 3–4 years.
Picking Parts

How to Choose a CPU

Intel vs AMD, core counts, and what specs actually matter for gaming and productivity.

  • Gaming — Prioritize single-core performance. Intel Core i5/i7 and AMD Ryzen 5/7 are both excellent for gaming. A 6-core CPU is plenty for most games.
  • Productivity/streaming — More cores help. Ryzen 9 or Intel Core i9 for video editing, 3D rendering, or heavy multitasking.
  • Platform matters — Pick AMD or Intel first, then the CPU. They use different sockets and motherboards. AMD AM5 supports future CPUs; Intel LGA1700/1851 is platform-specific.
  • TDP and cooler — Higher-end CPUs (125W+) need capable air or AIO coolers. Budget CPUs often include a stock cooler.
  • Don't overspend on CPU for gaming — A Ryzen 5 7600X paired with a strong GPU will outperform a Ryzen 9 with a weaker GPU in most games.
Picking Parts

How to Choose RAM

Speed, capacity, and whether DDR4 vs DDR5 matters for your build.

  • How much RAM? — 16 GB is the minimum for gaming today. 32 GB is recommended for gaming + streaming or light productivity. 64 GB+ for professional workloads.
  • Speed — DDR4-3600 is the sweet spot for AMD Ryzen. DDR5-6000 is the sweet spot for modern AMD platforms. Intel is less picky about speed.
  • Dual channel — Always use RAM in pairs (2 sticks) for ~10–15% performance improvement over a single stick.
  • DDR4 vs DDR5 — DDR5 is faster but more expensive. If you're on a modern Intel or AMD AM5 platform, DDR5 is worth it. Older platforms are DDR4 only.
  • Check motherboard QVL — Not all RAM kits are guaranteed compatible. Check the motherboard's qualified vendor list for your exact kit.
Picking Parts

How to Choose a PSU

Wattage, efficiency ratings, and why skimping on the power supply is a bad idea.

  • Calculate your needs — Add up CPU TDP + GPU TDP, then add 100–150W headroom. Use a PSU calculator for a precise estimate.
  • 80 Plus rating — Bronze is fine. Gold is better efficiency (less heat, lower electricity cost). Avoid unrated PSUs entirely.
  • Modular vs non-modular — Fully modular PSUs let you only connect cables you need, making cable management easier. Worth it for mid-range and above builds.
  • Brand matters — Corsair, Seasonic, EVGA, and Be Quiet! are reliable. Avoid cheap no-name PSUs — a failing PSU can damage other components.
  • Don't under-buy — A 650W PSU for an RTX 4090 build is dangerous. Headroom also extends PSU lifespan.

Overclocking & Tuning

CPU Overclocking: Getting Started

How to safely push your CPU beyond stock speeds for extra performance.

  • Requirements — You need a K/X-series CPU (Intel K or AMD X/XT), a Z or X motherboard, and a capable cooler. Stock coolers are not sufficient for OC.
  • Enter BIOS — Press DEL or F2 on boot. Look for CPU multiplier or Precision Boost Overdrive (AMD) settings.
  • Start conservatively — Increase the CPU multiplier by 1 step (e.g., 50x → 51x) and run a stability test before increasing further.
  • Watch temperatures — Keep CPU temps under 85–90°C under full load. Use HWiNFO or Ryzen Master to monitor.
  • Stability testing — Run Prime95 or Cinebench for at least 30 minutes. Crashes or BSOD mean the overclock is unstable — reduce multiplier or increase voltage slightly.
  • AMD Expo / Intel XMP — Enable this in BIOS to run RAM at its rated speed. This is technically overclocking and is safe for any XMP/Expo certified kit.
Overclocking & Tuning

GPU Overclocking with MSI Afterburner

Squeeze extra FPS out of your graphics card with a simple overclock.

  • Download MSI Afterburner — Free tool that works with any GPU brand (NVIDIA and AMD).
  • Start with power limit — Raise the Power Limit slider to 100% or its max. This alone can give a small free performance boost.
  • Core clock offset — Increase core clock by +50 MHz, run a benchmark (Unigine Heaven or 3DMark). If stable, increase by another +25 MHz and repeat.
  • Memory clock offset — Memory OC often gives more FPS than core OC in memory-bandwidth-limited scenarios. Start at +200 MHz and test.
  • Watch for artifacts — Black dots, flickering, or strange visual glitches mean the memory clock is too high. Reduce by 50 MHz.
  • Save your profile — Once stable, save the OC profile so it applies at startup.
  • Typical gains — A good GPU OC can yield 5–10% extra performance for free.
Overclocking & Tuning

Windows Performance Tweaks

Software-side optimizations to reduce latency and improve gaming performance.

  • Set Power Plan to High Performance — Control Panel → Power Options → High Performance. Prevents CPU from downclocking at idle.
  • Enable XMP/Expo in BIOS — If your RAM is running at 2133 MHz instead of 3600/6000, it's not running at its rated speed. Enable XMP in BIOS.
  • Disable unnecessary startup apps — Task Manager → Startup. Disable anything you don't need running at boot.
  • Enable Game Mode — Windows Settings → Gaming → Game Mode. Prioritizes game processes.
  • Keep GPU drivers updated — New drivers often include game-specific optimizations. Use GeForce Experience or AMD Software Adrenalin.
  • Enable Hardware-Accelerated GPU Scheduling (HAGS) — Windows Settings → Display → Graphics → Default graphics settings. Reduces CPU overhead for GPU commands.
  • Use NVIDIA DLSS or AMD FSR — If supported by your game, these upscaling technologies can nearly double FPS with minimal visual quality loss.
